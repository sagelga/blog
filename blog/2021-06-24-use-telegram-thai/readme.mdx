## ไม่ต้องดาวน์โหลด! วิธีใช้ภาษาไทยใน Telegram

Telegram นอกจากจะได้รับความนิยมในต่างประเทศและชุมชนผู้ใช้งานในประเทศไทยแล้ว แต่ก็ยังไม่รองรับภาษาไทยแบบเต็มตัว วันนี้เราจึงมาสอนวิธีการใช้ภาษาไทยบนแอพ Telegram ทั้ง iOS, Android, Windows, macOS, และ Linux โดยไม่ต้องดาวน์โหลดอะไรเพิ่มเติมอีกด้วย!

## ขั้นตอนในการเปลี่ยนภาษา

เข้าไปที่ลิงก์เว็บไซต์อย่างเป็นทางการของ Telegram

[Translations (telegram.org)](https://translations.telegram.org/thaith/)

จากหน้าต่าง ให้กดที่คำว่า “Use Telegram in Thai” ที่อยู่ด้านล่างในหัวข้อ “Actions”

![](https://cdn-images-1.medium.com/max/2048/0*Et22zdHZCdmZRYcN.png)

หากกดเข้าไปแล้ว หน้าเว็บไซต์ก็จะทำการเปิด Telegram บนอุปกรณ์ให้โดยอัตโนมัติ แต่หากเพื่อนๆ เปิดหน้าเว็บไซต์นี้และมีการแจ้งเตือนว่าต้องการเปิดด้วยอะไร ก็ให้เปิดลิงก์ผ่าน Telegram

![](https://cdn-images-1.medium.com/max/2048/0*C9DMX07eSMeHqgz7.png)

และเมื่อแอพ Telegram ที่อยู่บนอุปกรณ์เปิดขึ้นมาแล้ว ก็จะมีการถามว่าต้องการเปลี่ยนเป็นภาษาไทยหรือไม่ ให้ทำการกด ‘Change’ เพื่อเปลี่ยนเป็นภาษาไทย

![](https://cdn-images-1.medium.com/max/2048/0*oLqB7xtYzBCnggIe.png)

โดยคำแปลเหล่านี้ก็อาจจะไม่ครบถ้วนบ้าง เพราะการแปลภาษาไทยนี้ได้รับการแปลโดยอาสาสมัครในกลุ่มผู้ใช้งาน Telegram ที่ใช้ภาษาไทย ดังนั้นหากว่ามีคำแปลที่ผิดไป เราแนะนำให้ท่านเข้ามาให้คำแนะนำในการแปล หรือ มาเป็นส่วนหนึ่งในการแปลภาษาไทยในแอพ Telegram
